# praktikum-python-BR

## Ringkasan
Repo praktikum sinkronisasi Google Colab ↔ GitHub.

## Cara menjalankan
1. Buka notebook di folder `notebooks/`
2. Jalankan sel dari atas ke bawah

## Struktur folder
- `src/` : kode .py (fungsi/kelas)
- `notebooks/` : notebook praktikum
- `data/` : data kecil (hindari data besar/sensitif)
- `outputs/` : output penting (pilih yang perlu saja)

## Catatan data
Data besar disimpan terpisah (misal Google Drive); repo menyimpan skrip + instruksi.
